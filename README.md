# DorkSint

_OSINT tool for Dorks search - Yandex, Google, Bing, DuckDuckGo_


# 🔻**To install**:

- `git clone https://github.com/vertex-coder/DorkSint.git`


- `cd DorkSint`


- `pip install setup.py`


# 🔍 **Usage**:

- `dorksint {ur dork}`
