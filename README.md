# DorkSint


_OSINT tool for Dorks search - Yandex, Google, Bing, DuckDuckGo_


# 🔻**To install**:

- `git clone https://github.com/vertex-coder/DorkSint.git`


- `cd DorkSint`


- `pip install .`


# 🔍 **Usage**:

- Default search `dorksint {ur dork}`


- Search with PDF, WORD, EXCEL, DB files `dorksint -f {ur dork}`
